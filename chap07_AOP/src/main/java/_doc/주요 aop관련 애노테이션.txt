@Aspect : 애스펙트 클래스 선언
@Pointcut("execution(포인트컷 메서드)") : 포인트컷 메서드 지정
@Before("포인트컷 메서드") : 포인트컷 메서드 실행 전 호출되는 공통모듈 정의
@After("포인트컷 메서드") : 포인트컷 메서드 실행 후 호출되는 공통모듈 정의
@AfterReturning("포인트컷 메서드") : 포인트 컷 정상 실행 후 수행하는 공통모듈 정의
@Around("포인트컷 메서드") : 포인트컷 메서드 전과 후 수행하는 공통모듈 정의
@AfterThrowing("포인트컷 메서드") : 포인트 컷 예외 발생 시 수행하는 공통모듈 정의
